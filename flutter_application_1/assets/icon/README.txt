Coloca aquí tu imagen de icono para la app.

Nombre recomendado: app_icon.png
Tamaño recomendado: 1024x1024 (PNG, sin transparencia si prefieres evitar problemas en Android).

Luego de poner la imagen, ejecuta localmente:
  flutter pub get
  flutter pub run flutter_launcher_icons:main

Si no puedes ejecutar comandos ahora, la configuración ya está en `pubspec.yaml`.
